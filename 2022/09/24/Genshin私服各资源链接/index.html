服务器核心

预编译核心

预编译的核心可以直接在项目仓库的 Actions 内找到。

若使用预编译核心，则需要将以下三项资源放置于目录内：

  • keys
  • data
  • keystore.p12

自行编译核心

https://github.com/Grasscutters/Grasscutter
https://github.com/Grasscutters/Grasscutter/releases/
key:https://github.com/Grasscutters/Grasscutter/blob/development/keystore.p12
使用Git：从dev分支下载代码

git clone -b development https://ghproxy.com/https://github.com/Grasscutters/Grasscutter cd Grasscutter/
