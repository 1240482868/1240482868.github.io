安装git

本文主要讲解windows下的使用方法，linux大同小异

windows：直接百度Git

大陆地区可以用下面的地址
CNPM Binaries Mirror (npmmirror.com)
https://npm.taobao.org/mirrors/git-for-windows/
根据自己的系统来选择文件下载安装
安装的时候除了安装目录可以调整一下 其他默认就可以

使用

安装完毕之后在开始菜单应该可以看见这些

并且右键菜单也可以找到

在一个目录下右键打开Git Bash

新建仓库上传文件

git init   #将这个目录变成git可以管理的仓库


如果你的计算机设置了可以看见隐藏文件，那么你应该可以看见目录下出现了一个.git
尝试在目录下新建一个文件

之后我们在git里输入git add 文件名来告诉git把文件添加到仓库

执行之后应该不会有提示

提交或者修改来上传到Git

用git commit 告诉git 把文件提交到仓库 后面的参数-m 是本次提交的说明


上面的消息告诉我们，添加了一个文件 1 file changde，插入了一行内容 1 insertion
Add可以多次使用来添加很多文件 commit可以一次提交很多文件

现在我们修改仓库里的一个文件
然后运行git status


git status 可以让我们知道目前仓库的状态，上面的信息告诉我们
Readme.txt被修改了 ，但是还没有提交
未完
